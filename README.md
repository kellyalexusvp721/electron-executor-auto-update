**Is Electron Executor tied to Windows?**  
Yes. The project is described as a Windows desktop app for Roblox scripting use.

**Will it work after Roblox updates?**  
Compatibility may shift when Roblox updates its client or underlying behavior. The included auto-update engine is meant to help the app stay in step with newer builds.

**How is it different from a basic script loader?**  
It goes beyond a single-run launcher by including a script hub, persistent queueing, batch execution, and a built-in debugger.

**Is there any account risk?**  
Third-party script tools can come with user responsibility and platform-specific terms. Review your own usage and policies before running any tool with Roblox.

**Where are scripts stored?**  
The product includes persistent queue behavior, so local settings and script selections may be kept on the device depending on how you configure it.
